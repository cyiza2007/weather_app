# Weather Web Application (Level 3 SWD)

## 📌 Description
This is a responsive weather web application that allows users to search for any location
(city, town, or region) and view the current weather conditions.

The app displays:
- Temperature (°C)
- Weather condition + icon
- Humidity (%)
- Wind speed (km/h)
- Feels-like temperature (°C)

It also includes:
- Loading indicator
- Error handling
- LocalStorage (remembers last searched location)
- Geolocation button (detect current location)

---

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeatherMap API

---

## 🌍 Weather API Used
OpenWeatherMap Current Weather Data API

### How to Get API Key
1. Go to https://openweathermap.org/
2. Create a free account
3. Login to your account
4. Go to API Keys section
5. Copy your API key
